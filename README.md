# CV / Resume Maker Project

## 📌 Project Overview
The **CV / Resume Maker** is a web-based project developed using **HTML, CSS, and JavaScript** as part of my first semester coursework. It allows users to create and customize a professional resume using a ready-made template.

Users can easily edit their personal information and instantly see changes in real time.

---

## ⚙️ Features

- 📝 Pre-designed resume template
- ✏️ Live editing of:
  - Name
  - Profile picture
  - Skills
  - Contact details
  - Other personal information
- 🔗 Clickable links (social profiles, email, etc.)
- 🌙 Dark theme UI design
- 🖨️ Print resume option
- 📥 Download resume option
- ⚡ Instant preview of changes

---

## 💻 Technologies Used

- HTML (structure)
- CSS (styling + dark theme)
- JavaScript (functionality and dynamic updates)

> Note: HTML, CSS, and JavaScript are written in a single file (inline approach).

---

## 🎯 Project Purpose
The main purpose of this project is to help users quickly generate a professional CV using a simple and interactive interface without needing any design skills.

It also demonstrates core web development concepts learned in the first semester.

---

## 🚀 How It Works
1. Open the project in a browser  
2. Edit the template fields (name, skills, etc.)  
3. Upload profile picture  
4. Preview changes instantly  
5. Click **Print** or **Download** to save CV
